# IBM Data Science Capstone: SpaceX Falcon 9 Landing Prediction

A data science project analysing SpaceX Falcon 9 launch data to predict first-stage landing success. Completed as the capstone for the IBM Data Science Professional Certificate.

[**View the full presentation**](./ds-capstone-coursera.pdf)

## Project Overview

This project applies the full data science methodology to predict whether a Falcon 9 first stage will successfully land, enabling cost estimation for launches.

**Skills demonstrated:** Data collection, data wrangling, exploratory data analysis, SQL, data visualisation, machine learning classification.
